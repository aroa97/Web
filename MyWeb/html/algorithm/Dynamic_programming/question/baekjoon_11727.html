백준 11727. 2×n 타일링 2
  • https://www.acmicpc.net/problem/11727

  • 문제 :

    2×n 직사각형을 1×2, 2×1과 2×2 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.

    아래 그림은 2×17 직사각형을 채운 한가지 예이다.

  • 입력 :

    첫째 줄에 n이 주어진다. (1 ≤ n ≤ 1,000)

  • 출력 :

    첫째 줄에 2×n 크기의 직사각형을 채우는 방법의 수를 10,007로 나눈 나머지를 출력한다.

  • 풀이 :

    [백준 11726. 2×n 타일링]와 유사한 문제이다. 2×2 타일이 추가가 된 것 뿐이다.
    n이 1일 때 방법의 수는 1개다.
    n이 2일 때 방법의 수는 3개다.



    n이 3부터는 아래 그림 모양을 붙혀 방법의 수를 구한다.

    세로로 세워진 두 개의 2×n 크기의 직사각형의 경우 중복되는 모양이 발생하므로 가로로 눕혀진 두 개의 2×n 크기의 직사각형과 2×2 크기의 정사각형을 사용한다.

    점화식은 f(n) = f(n - 1) + f(n - 2) + f(n - 2) 다.
    하지만 문제에서 10007로 나눈 나머지를 출력하라고 했으므로 f(n) = (f(n - 1) + 2 * f(n - 2)) % 10007 로 점화식을 변경한다.
