백준 11725. 트리의 부모 찾기
  • https://www.acmicpc.net/problem/11725

  • 문제 :

    루트 없는 트리가 주어진다. 이때, 트리의 루트를 1이라고 정했을 때, 각 노드의 부모를 구하는 프로그램을 작성하시오.

  • 입력 :

    첫째 줄에 노드의 개수 N (2 ≤ N ≤ 100,000)이 주어진다. 둘째 줄부터 N-1개의 줄에 트리 상에서 연결된 두 정점이 주어진다.

  • 출력 :

    첫째 줄부터 N-1개의 줄에 각 노드의 부모 노드 번호를 2번 노드부터 순서대로 출력한다.

  • 풀이 :

    1이 최상위에 위치하고 2부터 N까지의 부모 노드를 출력해야하는 문제이다.
    각 노드의 부모 노드를 저장할 배열을 만들고 BFS를 통해 부모 노드를 찾기로 했다.

    1이 최상위 부모 노드이므로 1부터 BFS 탐색을 시작하고 그 과정에서 탐색된 노드의 부모 노드는 1이 된다.

    1 노드의 탐색이 끝나면 탐색된 노드의 탐색을 시작한다. 이 과정을 반복하면 모든 노드의 부모 노드를 알 수 있게 된다.


    모든 탐색이 끝나면 2부터 N까지의 부모 노드를 차례대로 출력한다.
