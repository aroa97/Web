백준 13015. 별 찍기 - 23
  • https://www.acmicpc.net/problem/13015

  • 문제 :

    예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.

  • 입력 :

    첫째 줄에 N(2 ≤ N ≤ 100)이 주어진다.

  • 출력 :

    규칙대로 별을 출력한다.

  • 풀이 :

    문제의 예제는 다음과 같다.



    우선 X자 모양의 위와 아래에 N의 값과 같은 수의 별이 찍혀있다.


    가운데 맨 위에는 N * 2 - 3만큼의 빈칸이 있으며 행이 증가할수록 2개씩 적어지고 있다.


    왼쪽 두번째 줄부터는 빈칸이 하나씩 증가하고 있다.


    빈칸 다음 N 값만큼의 공간에는 처음과 끝에 별이 찍혀있으며 N - 2 만큼의 빈칸이 있다. 이러한 모양이 왼쪽, 오른쪽 하나씩 있다.


    가운데는 겹치는 부분이 생기므로 주의한다.


    이러한 규칙을 신경쓰면서 코드를 작성, 밑의 나머지 부분은 빈칸의 규칙을 거꾸로 적용하면 된다.
