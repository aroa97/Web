백준 10817. 세 수
  • https://www.acmicpc.net/problem/10817

  • 문제 :

    세 정수 A, B, C가 주어진다. 이때, 두 번째로 큰 정수를 출력하는 프로그램을 작성하시오.

  • 입력 :

    첫째 줄에 세 정수 A, B, C가 공백으로 구분되어 주어진다. (1 ≤ A, B, C ≤ 100)

  • 출력 :

    두 번째로 큰 정수를 출력한다.

  • 풀이 :

    정수 A, B, C가 주어지고 이 중에서 두 번째로 큰 정수를 출력하는 문제이다.
    if문을 사용해 세 수를 비교해서 출력해도 되지만 세 수를 합한 값에서 가장 작은 수랑 가장 큰를 빼도 답을 구할 수 있기 때문에 min과 max 함수를 사용해서 답을 구하기로 했다.



    데이터를 입력 받은 뒤 세 수의 합을 구하고 min과 max 함수를 사용해 가장 작은 수와 가장 큰 수를 구하고 입력 받은 데이터를 모두 합한 값에서 앞에서 구한 가장 작은 수와 가장 큰 수를 뺀 답을 출력.
