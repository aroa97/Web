백준 2852. NBA 농구
  • https://www.acmicpc.net/problem/2852

  • 문제 :

    동혁이는 NBA 농구 경기를 즐겨 본다. 동혁이는 골이 들어갈 때 마다 골이 들어간 시간과 팀을 적는 이상한 취미를 가지고 있다.

    농구 경기는 정확히 48분동안 진행된다. 각 팀이 몇 분동안 이기고 있었는지 출력하는 프로그램을 작성하시오.

  • 입력 :

    첫째 줄에 골이 들어간 횟수 N(1<=N<=100)이 주어진다. 둘째 줄부터 N개의 줄에 득점 정보가 주어진다. 득점 정보는 득점한 팀의 번호와 득점한 시간으로 이루어져 있다. 팀 번호는 1 또는 2이다. 득점한 시간은 MM:SS(분:초) 형식이며, 분과 초가 한자리 일 경우 첫째자리가 0이다. 분은 0보다 크거나 같고, 47보다 작거나 같으며, 초는 0보다 크거나 같고, 59보다 작거나 같다. 득점 시간이 겹치는 경우는 없다.

  • 출력 :

    첫째 줄에 1번 팀이 이기고 있던 시간, 둘째 줄에 2번 팀이 이기고 있던 시간을 출력한다. 시간은 입력과 같은 형식(MM:SS)으로 출력한다.

  • 풀이 :

    두 팀 중 어느 한 팀이 득점하면서 이기는 상황이 되었을 때랑 두 팀 중 어느 한 팀이 득점하면서 동점이 되는 상황이 되었을 때만 신경쓰면 된다.

    입력받은 데이터를 차례대로 확인한다.

    1번 팀이 득점하면서 이기는 상황이 되었다. 경기 시간 48분에 득점한 시간을 뺀 값을 1팀이 이기고 있던 시간에 더한다. 편의상 분을 초로 변경해서 계산한 뒤 나중에 답을 출력할 때 원래대로 변경한다.


    1번 팀이 득점했고 1번 팀이 이기는 상황은 변하지 않았다.


    2번 팀이 득점했지만 1번 팀이 이기고 있는 상황은 변하지 않았다.


    2번 팀이 득점하면서 동점이 된 상황이 되었다. 1번 팀이 이기는 상황이 된 시간부터 동점이 된 시간까지만 1번 팀이 이기고 있던 시간이므로 경기 시간에서 동점이 된 시간을 뺀 시간만큼 1번이 이기고 있던 시간을 뺀다.


    2번이 득점하면서 이기는 상황이 되었다. 경기 시간 48분에 득점한 시간을 뺀 값을 2팀이 이기고 있던 시간에 더한다.


    모든 데이터를 확인했고 이러한 과정을 통해 나온 값을 분, 초로 변경해서 문제에서 원하는 출력방식으로 출력한다.
